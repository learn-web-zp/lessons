CSS: основы

30 селекторов к прочтению

  • селекторы
    • элемент (имя тега): a, li, ul, …

  • вложенные селекторы
    • x y
    • #x y
    • #x .y
    • .x #y
    • …

    • x#y z
    • …
  • особые селекторы
    • *

  • псевдоклассы

    • :hover
    • :checked

    • :after
    • :first-child | :last-child
